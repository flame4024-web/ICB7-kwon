# 📧 네이버 메일 자동 관리 가이드 (최종본)

이 문서는 `mail_manager.py`를 사용하여 네이버 메일함을 효율적으로 자동 관리하는 방법과 주요 업데이트 사항을 설명합니다.

## 📋 핵심 자동화 기능

1. **메일함 전체 일괄 읽음 처리**: 
   - '받은 메일함'뿐만 아니라 모든 사용자 메일함의 읽지 않은 메일을 '읽음' 상태로 변경합니다.
   - **읽음 예외**: 중요한 메일이 담긴 **'개인'** 및 **'중앙세무회계'** 메일함은 읽음 처리 대상에서 **자동으로 제외**되어 원래 상태를 유지합니다.
2. **스팸 및 휴지통 영구 삭제**: 
   - 'Junk' 및 'Deleted Messages' 폴더를 비워 저장 공간을 즉각 확보합니다.
3. **오래된 메일 이관 (6개월 기준)**: 
   - **6개월(180일) 이상 경과**된 메일을 전수 조사하여 휴지통으로 이동시킵니다.
   - **예외 폴더 보호**: `'구독/개인'` 및 `'어세스타/중앙세무회계'` 폴더는 한 건의 메일도 건드리지 않고 안전하게 보존합니다.

## 🛠️ 기술적 고도화 사항 (업데이트)

- **폴더명 인용부(`" "`) 처리**: 'Deleted Messages'와 같이 공백이 포함된 특수 폴더명도 오류 없이 인식하도록 인용부 처리 로직을 강화했습니다.
- **휴지통/스팸함 자동 식별**: 서버의 IMAP 속성(`\Trash`, `\Junk`)을 분석하여 네이버 메일함 설정에 상관없이 정확한 대상 폴더를 자동으로 찾아냅니다.
- **대용량 이관 지원**: '뉴스'(697건), '구독'(323건) 등 대규모 메일함도 누락 없이 안정적으로 이관할 수 있도록 설계되었습니다.

## 🚀 실행 결과 및 사용법

### 1. 환경 설정 확인
`c:\mskwon_data\.venv\.env` 파일 내에 네이버 계정과 애플리케이션 비밀번호(16자리)가 입력되어 있는지 확인하세요.

### 2. 실행 명령어 (터미널)
```powershell
# UTF-8 엔코딩 적용 후 실행
chcp 65001
uv run python "c:\mskwon_data\auto\3. 메일정리\mail_manager.py"
```

## ⚠️ 주의 사항
- 6개월 경과 메일은 휴지통으로 우선 이동됩니다. 영구 삭제를 원치 않는 중요한 메일은 미리 '개인' 폴더나 예외로 지정된 폴더로 이동시켜 두는 것을 권장합니다.
- 예외가 필요한 폴더가 더 생길 경우, 스크립트 내 `exclude_folders` 리스트에 추가하여 간편하게 관리할 수 있습니다.
