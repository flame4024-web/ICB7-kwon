# 슬랙 API 설정 가이드 (MVP 단계)

이 가이드는 안티그래비티와 연결할 슬랙 비서 앱을 처음부터 만드는 과정을 안내합니다.

## 1단계: 슬랙 API 사이트 접속 및 앱 생성
1. [api.slack.com/apps](https://api.slack.com/apps)에 접속합니다. (로그인이 필요할 수 있습니다.)
2. 오른쪽 상단의 **[Create New App]** 버튼을 클릭합니다.
3. 팝업창에서 **"From scratch"**를 선택합니다.
4. **App Name**: `AntiGravity_Bot` (또는 원하는 이름)을 입력합니다.
5. **Development Slack Workspace**: 앱을 설치할 내 워크스페이스를 선택합니다.
6. **[Create App]** 버튼을 클릭합니다.

## 2단계: 필요한 권한(Scopes) 부여
메시지를 읽고(요약), 알림을 보내기 위해 다음 설정을 진행합니다.

1. 왼쪽 사이드바 메뉴에서 **[OAuth & Permissions]**를 클릭합니다.
2. 페이지를 중간쯤 내리면 **"Scopes"** 섹션이 나옵니다.
3. **"Bot Token Scopes"** 아래에 있는 **[Add an OAuth Scope]** 버튼을 클릭하여 아래 항목들을 하나씩 추가합니다.

### 필수로 추가해야 할 Scopes (총 6개)
*   `chat:write`: 메시지를 보낼 수 있는 권한 (알림 전송용)
*   `channels:history`: 공개 채널의 메시지를 읽을 수 있는 권한 (요약용)
*   `groups:history`: 비공개 채널의 메시지를 읽을 수 있는 권한
*   `im:history`: 1:1 DM 메시지를 읽을 수 있는 권한
*   `mpim:history`: 다수 요원 DM 메시지를 읽을 수 있는 권한
*   `users:read`: 사용자 리스트 정보를 볼 수 있는 권한

## 3단계: 앱 설치 및 토큰 발급
1. Scopes 설정을 마쳤다면, 페이지 상단의 **[Install to Workspace]** 버튼을 클릭합니다. (초록색 체크 표시가 보이면 권한 설정이 완료된 것입니다.)
2. 권한 허용 화면이 나오면 **[Allow]** (또는 허용)을 클릭합니다.
3. 설치가 완료되면 **"Bot User OAuth Token"**이 생성됩니다.
    *   이 토큰은 `xoxb-`로 시작하며, 안티그래비티가 슬랙에 접속할 때 사용하는 비밀번호와 같습니다.
    *   **주의**: 이 토큰을 외부에 노출하지 마세요.

## 4단계: (중요) 슬랙 채널에 앱 초대하기
앱만 만든다고 바로 메시지를 읽을 수 있는 것은 아닙니다.
1. 슬랙 PC 또는 모바일 앱을 엽니다.
2. 요약하고 싶은 채널에 들어갑니다.
3. 채널 본문에서 `/invite @AntiGravity_Bot`(설정한 앱 이름)을 입력하여 엔터를 칩니다.
4. 이제 앱이 해당 채널의 대화를 볼 수 있는 준비가 되었습니다!

---
**성공 확인**: 'OAuth & Permissions' 메뉴 상단에 `Bot User OAuth Token`이 보이고 바로 아래에 초록색 체크 표시가 보인다면 1단계 완료입니다!

## 5단계: 안티그래비티 슬랙 MCP 주요 기능 및 사용법

안티그래비티 슬랙 MCP를 연동하면 아래와 같은 작업을 수행할 수 있습니다. 각 기능별 추천 프롬프트를 참고하여 AI 비서를 활용해 보세요.

### 1. 주요 기능 (권한 기반)
*   **메시지 전송 및 관리 (`chat:write`)**: 채널 메시지 및 DM 발송, 스레드 답변, 이모지 반응 추가
*   **대화 맥락 파악 (`history` 계열)**: 공개/비공개 채널 및 DM의 최근 대화 내용 조회 및 요약
*   **워크스페이스 정보 조회 (`channels:read`, `users:read`)**: 전체 채널 목록 및 멤버 프로필 확인
*   **파일 공유 확인 (`files:read`)**: 대화 중 공유된 파일 정보 조회

### 2. 활용 프롬프트 예시

안티그래비티 대화창에서 명령 시 **`@mcp:slack:`** 접두사를 사용하면 더 정확하게 작동합니다.

| 작업 카테고리 | 프롬프트 예시 (Copy & Paste) |
| :--- | :--- |
| **대화 내용 요약** | `@mcp:slack:#general 채널의 최근 대화 5개를 읽고 어떤 내용인지 요약해줘.` |
| **특정 멤버에게 DM** | `@mcp:slack:권민섭님에게 '안티그래비티 슬랙 테스트 성공입니다!'라고 메시지 보내줘.` |
| **스레드 맥락 파악** | `@mcp:slack:이 스레드에 달린 답글들을 모두 읽고 결정된 사항을 정리해줘.` |
| **리액션 추가** | `@mcp:slack:방금 메시지에 :white_check_mark: 이모지 반응을 달아줘.` |
| **멤버 검색** | `@mcp:slack:워크스페이스 멤버 중에서 '기획'팀 담당자가 누구인지 찾아줘.` |
| **파일 전송 확인** | `@mcp:slack:#개발 채널에 오늘 오전 중 올라온 파일이 있는지 확인해줘.` |

### 💡 활용 팁
*   **초대 필수**: 봇이 대화를 읽으려면 해당 채널에 반드시 초대되어 있어야 합니다. (`/invite @앱이름`)
*   **자연어 명령**: "민섭님에게 보내줘"와 같이 이름만 언급해도 AI가 멤버 리스트에서 찾아 메시지를 보냅니다.
*   **복합 명령**: "최근 대화 요약하고 그 결과를 보고서 채널에 공유해줘"와 같은 연속 작업도 가능합니다.
